# Financial Management App

This is a comprehensive financial management application built with Flutter and Firebase. It provides users with a powerful set of tools to track their income and expenses, plan their finances, and achieve their financial goals.

## Features

- **User Authentication:** Secure sign-up and sign-in with email and password.
- **Transaction Management:** Easily add, edit, and delete income and expense transactions.
- **Transaction Tagging:** Categorize transactions with specific tags for more granular financial insights.
- **Financial Planning:** A dedicated planning section with a calendar view to visualize upcoming transactions.
- **Data Visualization:** Interactive charts and graphs to visualize income, expenses, and savings progress.
- **Modern UI:** A clean, modern, and intuitive user interface built with Material Design principles.

## Architecture

The application is built using a clean and scalable architecture, with a clear separation of concerns between the different layers of the application.

- **State Management:** BLoC (Business Logic Component) for predictable and maintainable state management.
- **Dependency Injection:** `get_it` for managing dependencies and promoting modularity.
- **Feature-based Folder Structure:** The project is organized by feature, making it easy to navigate and maintain.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/financial-management-app.git
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the app:**
   ```bash
   flutter run
   ```
